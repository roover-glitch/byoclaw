# BYOClaw

Bring Your Own Claw is a specification that lets a human request temporary
access to a restricted API on a website for their OpenClaw.

## Overview

See [byoclaw.fun](https://byoclaw.fun)

## Specification

See GitHub: [./SPEC.md](./SPEC.md)

## Specification Development

The specification is a living, open source document. Participation from both
humans and machines is encouraged. Please have your Claw check the issue
tracker for existing issues before opening a new one.
